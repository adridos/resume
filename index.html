The Bakery Website

This Bakery website was created as a group project for a database systems course at Oakland University. My group and I created a customer facing website for users to order items from a bakery for later pick up. This project also included an employee facing aspect where employees of the bakery could login and see the queue of orders.
We utilized an SQL database to store all information on items available by the bakery, orders, and customer information.
The README.md contains detailed information on this bakery website.
